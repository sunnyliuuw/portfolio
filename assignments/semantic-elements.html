Semantic Elements

Semantic elements are the tags with clear meanings for both the browsers and developers. For example, article means this part is the article, div tells nothing about the content but a block for something. Semantic elements help people read, interpret, and understand the structure of your webpage easily. Also, semantic elements help organize webpage easily, instead of div id = abc.
